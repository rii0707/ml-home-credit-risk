# scripts

One-off tools run from the **project root** (not from `src/`).

| Script | Description |
|--------|-------------|
| `download_home_credit_csvs.py` | Download Kaggle *Home Credit Default Risk* CSVs into `data/raw/`. See `config/` for authentication. |

Example:

```bash
python scripts/download_home_credit_csvs.py
```
